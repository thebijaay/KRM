## About This Project

This project is a static website built with HTML, CSS, and JavaScript. It showcases the portfolio, skills, and services of Jay Prasad Majhi. It also includes a functional user authentication system and an admin dashboard, with a Python backend (`server.py`) to handle dynamic features.

## Key Features

- **Portfolio Showcase:** Detailed sections for "Latest Works," "About Me," "Services," and "Testimonials."
- **User Authentication:** Secure login, registration, and OTP verification pages.
- **Admin Dashboard:** A private dashboard for system administration and user management.
- **Responsive Design:** The website is designed to be fully responsive across desktops, tablets, and mobile devices.
- **Modern Tech Stack:** Built with modern frontend technologies and Tailwind CSS.
